# Echo Guardian iOS Native Package v0.4

This Swift Package is a native implementation scaffold.

It includes:

- Keychain-backed proof-of-possession signing prototype
- minimized sensor observation model
- plain-language status view model
- guarded local-outbox delivery provider boundary

Secure Enclave must only be claimed where the platform actually supports the operation. Unsupported primitives must be protected using Keychain or vetted libraries and labeled accurately.
