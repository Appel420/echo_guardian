# Echo Guardian Android Native Module v0.4

This module is a Kotlin/Android scaffold for native implementation.

It includes:

- Android Keystore proof-of-possession signing prototype
- minimized sensor observation model
- plain-language status view model
- guarded local-outbox delivery provider boundary

Production apps must preserve the product laws: no hidden behavior, no default private-space monitoring, no raw sensor retention, no emergency-services routing by default, and no compliance/medical claims without review.
