# Echo Guardian Production Foundation v0.5 Release Notes

## Release

Native Runtime Implementation Start

## Added

- iOS Swift runtime bridge from minimized sensor observation to core-compatible JSON.
- Android Kotlin runtime bridge from minimized sensor observation to core-compatible JSON.
- Native audit append interface for local JSONL event output.
- Native policy/status screen models.
- Keychain / Keystore proof-of-possession persisted-key test scaffolds.
- Guarded contact delivery provider test harness documentation.
- Native accessibility UI skeleton.
- CI native build validation without soft-fail placeholders.
- v0.5 SBOM and release readiness report.

## Production Laws Preserved

- No hidden behavior.
- No cloud dependency for the core loop.
- No private-space monitoring by default.
- No raw sensor retention by default.
- No emergency-services path by default.
- No silent telemetry.

## Still Not Approved For

- GDPR/global-compliance claims.
- Medical-device claims.
- Emergency-system certification claims.
- Clinical deployment.
- Silent enterprise/MDM control.
