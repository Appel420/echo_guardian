# Echo Guardian Production Foundation v0.6 Release Notes

## Release Focus

Native Runtime Hardening.

## Added

- iOS CoreMotion permission/status implementation scaffold.
- Android sensor permission/status implementation scaffold.
- Native audit append to core audit-chain bridge.
- Native baseline update integration.
- Native severity classification integration.
- Provider adapter delivery-status persistence.
- Explicit native export package creation.
- CI artifact enforcement for v0.6.
- Expanded native security tests for private-space, raw-data, egress, and audit-bypass blocking.

## Validation Boundary

v0.6 is approved only as a native runtime hardening baseline. It is not approved for compliance claims, clinical deployment, medical-device claims, emergency-system certification claims, or silent enterprise/MDM control.
