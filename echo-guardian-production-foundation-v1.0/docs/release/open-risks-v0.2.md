# Open Risks v0.2

| Risk | Severity | Owner | Mitigation |
|---|---:|---|---|
| False positive rate | High | Safety Engineering | Controlled real-world validation and conservative threshold tuning |
| False negative rate | High | Safety Engineering | Sensor diversity, incident review, and failure-injection testing |
| Mobile background execution | High | Platform Engineering | Native iOS/Android architecture and OS permission review |
| Secure key lifecycle | High | Security Engineering | Platform-backed keys, rotation, migration, and recovery boundaries |
| Accessibility validation | High | Product / Accessibility | User testing with elderly and disabled users |
| Global compliance | High | Compliance | Jurisdiction-specific legal and privacy review |
| Live provider reliability | Medium | Platform / Escalation | Delivery adapters, retries, status receipts, and failure audit |
