# v1.0 Foundation Readiness Gate

The v1.0 foundation readiness gate is evidence-bound.

## Required release-blocking evidence

- Trusted Gradle wrapper evidence
- Real iOS physical-device evidence
- Real Android physical-device / local native test evidence
- Verified signed SBOM attestation
- Completed external security review
- Completed privacy review
- Passed clinical/legal claim-blocking review
- No HIPAA/GDPR claims
- No medical-device claims
- No emergency-system claims
- No clinical deployment claims
- No silent enterprise/MDM control

## Decision rule

If any release-blocking item is missing, rejected, or incomplete, the result is `blocked`.
