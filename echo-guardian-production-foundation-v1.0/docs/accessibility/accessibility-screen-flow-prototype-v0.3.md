# Accessibility Screen-Flow Prototype v0.3

## Objective

Define the first accessible local screen flow for a child, elderly person, caregiver, or non-technical user.

## Screens

1. Home Status
   - Shows monitoring on/off.
   - Shows active spaces.
   - Shows active signal types.
   - Shows who controls each setting.

2. What Are You Doing?
   - Plain-language answer.
   - No technical jargon by default.

3. Safety Check Prompt
   - Voice, text, and large-touch response options.
   - Clear fallback if one mode fails.

4. Alert Preview
   - Shows exactly what a contact would receive.
   - Clearly says emergency services are not contacted by default.

5. Health / Degraded State
   - Explains what is not working and what still works.

6. Audit Review
   - Plain-language recent events.
   - Advanced verification available separately.

## Product Rule

Simple on the outside. Strong underneath. Nothing hidden.
