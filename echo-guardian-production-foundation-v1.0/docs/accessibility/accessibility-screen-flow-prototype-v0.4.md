# v0.4 Accessibility Screen-Flow Prototype

The interface must be understood by a child, an elderly person, a caregiver, and a non-technical user.

## Main status screen

Primary question: **What are you doing?**

Plain response:

> Echo Guardian monitoring is on. It is watching the living room for safety changes. It uses movement and nearby-device signals. Control is user controlled. Everything needed for this local status view is working.

## Monitoring off screen

> Echo Guardian monitoring is off. No rooms are being watched right now.

## Degraded state

> Echo Guardian can still run, but one part needs attention. Safety checks may be less accurate until this is fixed.

## Alert preview

> If you do not respond, Echo Guardian will tell your safety contact: "No response after safety confirmation prompt. Please check on the user." No sensitive details are included. Emergency services are not contacted by default.

## Accessibility modes

- Voice
- Large touch
- Text
- Screen reader
- High contrast
- Plain-language status

No critical flow may depend on one interface mode only.
