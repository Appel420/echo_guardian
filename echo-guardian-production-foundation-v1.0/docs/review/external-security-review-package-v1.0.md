# v1.0 External Security Review Package

The v1.0 foundation readiness gate requires completed external security review evidence.

## Required materials

- `SECURITY.md`
- cryptographic design notes
- native device evidence materials
- audit verification tests
- hardening and release tests
- SBOM and signed SBOM attestation

## Gate rule

A package being prepared is not the same as review completion. The final gate requires reviewer identity, completion date, pass result, and no unresolved blockers.
