# v1.0 Privacy Review Package

The v1.0 foundation readiness gate requires completed privacy review evidence.

## Required materials

- `PRIVACY.md`
- data retention rules
- no raw sensor retention policy
- export consent rules
- transcript redaction proof
- no hidden telemetry evidence
- no silent enterprise/MDM control evidence

## Gate rule

The privacy review is blocked if hidden telemetry, raw retention, automatic export, or silent control is introduced.
