# v1.0 iOS Physical-Device Evidence

Simulator evidence is not sufficient for the v1.0 foundation readiness gate.

## Required iOS evidence

- Physical iOS device model and OS version.
- App build identifier.
- Permission/status screen evidence.
- Private-space blocking evidence.
- Raw-data blocking evidence.
- Signed native audit checkpoint verification.
- Accessibility test evidence capture.
- Execution log hash.

## Blocked claims

This evidence does not approve HIPAA, GDPR, medical-device, emergency-system, clinical deployment, or silent MDM claims.
