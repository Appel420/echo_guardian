# v1.0 Trusted Gradle Wrapper Evidence

Echo Guardian v1.0 foundation readiness requires trusted Android Gradle wrapper evidence.

## Required evidence

- `gradle/wrapper/gradle-wrapper.properties` pins Gradle `8.7`.
- `gradle/wrapper/gradle-wrapper.jar` is generated locally from a trusted Gradle installation.
- The wrapper JAR SHA-256 is recorded.
- Android local tests are run with:

```sh
cd native/android
./gradlew :echo-guardian-core:testDebugUnitTest
```

## Product-law rule

Do not invent, hand-edit, or fabricate `gradle-wrapper.jar`. If the JAR is missing, the v1.0 foundation readiness gate remains blocked.
