# Echo Guardian v0.6 Native Runtime Hardening

Status: implementation scaffold with enforced product-law boundaries.

## Scope

v0.6 hardens the v0.5 native runtime start by adding concrete native permission/status boundaries, native-to-core audit append behavior, baseline and severity integration hooks, guarded delivery-status persistence, explicit native export package creation, and native security regression coverage.

## Implemented Native Boundaries

- iOS CoreMotion permission/status reader scaffold.
- Android sensor permission/status reader scaffold.
- Swift and Kotlin native-to-core bridge checks.
- Python core-compatible `NativeRuntimeHardeningRunner`.
- Native audit append facade that writes to the core audit chain.
- Native baseline update integration.
- Native severity classification integration.
- Local provider delivery-status persistence.
- Explicit local export package creation.

## Product Laws Preserved

- No cloud dependency for the core safety loop.
- No hidden telemetry.
- No private-space monitoring by default.
- No raw sensor retention.
- No emergency-services routing by default.
- No sensitive details in guarded provider delivery-status persistence.
- No native observation accepted unless it is minimized and core-compatible.

## Runtime Cycle

1. Native permission/status is checked and audited.
2. Native observation envelope is validated.
3. Observation is converted into a core `DeviceSignalSample` without raw sample data.
4. Core sensor ingestion emits a minimized observation record.
5. Baseline learning updates aggregate local state.
6. Severity classifier produces an explainable deterministic decision.
7. Guarded provider delivery status can be persisted locally.
8. Native export can be explicitly generated with hashes.
9. Core audit chain is verified before cycle result is returned.

## Not Approved

v0.6 still does not approve HIPAA compliance claims, GDPR/global-compliance claims, medical-device claims, emergency-system certification claims, clinical deployment, or silent enterprise/MDM control.
