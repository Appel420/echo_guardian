# v1.0 Android Local / Native Test Evidence

Android local native test evidence is release-blocking.

## Required Android evidence

- Trusted Gradle wrapper JAR generated from the pinned Gradle version.
- Physical Android device model and OS version for device evidence.
- Local command output from `./gradlew :echo-guardian-core:testDebugUnitTest`.
- Private-space blocking evidence.
- Raw-data blocking evidence.
- Signed audit checkpoint verification.
- Accessibility evidence.

## No soft fail

The test command must not use `|| true`, `continue-on-error`, `allow_failure`, or any soft-fail mechanism.
