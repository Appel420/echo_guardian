# Trusted Gradle Wrapper JAR Generation Instructions v0.9

Echo Guardian does not vendor or invent `gradle-wrapper.jar` in this package. For reproducible local Android execution, generate the wrapper from a trusted local Gradle installation and record evidence.

## Required command

```sh
gradle wrapper --gradle-version 8.7 --distribution-type bin
./gradlew --version
./gradlew :echo-guardian-core:testDebugUnitTest
```

## Required evidence

- `gradle/wrapper/gradle-wrapper.properties` pinned to the expected distribution URL.
- SHA-256 for the generated `gradle-wrapper.jar`.
- Local test execution log from `./gradlew :echo-guardian-core:testDebugUnitTest`.
- Confirmation that no `continue-on-error`, `|| true`, `soft-fail`, or `allow_failure` behavior was used.

## Rule

Android native test execution is release-blocking for v1.0 foundation readiness.
