# Android Local Test Execution Evidence Format v0.9

Required fields:

- command: `./gradlew :echo-guardian-core:testDebugUnitTest`
- gradle wrapper present: true
- gradle wrapper JAR SHA-256: recorded after trusted generation
- tests passed: true
- soft-fail used: false
- execution log hash
- release blocking: true

Evidence created from emulator-only or soft-failed execution is not acceptable for release-candidate approval.
