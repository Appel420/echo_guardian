# Echo Guardian

Echo Guardian is a local-first safety monitoring foundation with production-oriented controls for consent, policy, audit, escalation, and release evidence.

## Repository layout

- `echo-guardian-production-foundation-v1.0/` — main project source, docs, tools, and tests
- `echo-guardian-production-foundation-v1.0/README.md` — primary project documentation

## Quick start

```bash
cd echo-guardian-production-foundation-v1.0
python -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
make test
```

## Notes

- The project is designed around local execution and auditability.
- See the README inside `echo-guardian-production-foundation-v1.0/` for the full production-foundation overview.
